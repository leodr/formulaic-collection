> Collection of formulas written in Markdown and displayed with Next.js and MDX.

### 🏠 [Homepage](https://github.com/leodr/formulaic-collection#readme)

## Prerequisites

-   npm >=5.5.0
-   node >=10.4.0

## Install

```sh
yarn
```

## Usage

```sh
yarn dev
```
